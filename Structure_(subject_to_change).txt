Database: 
will store the set up for the chores, users, and their chore streak
set up with sqlite 
integrate with Flask

firebase:
admin and user side

admin: can send out new chores to the users
can track users and their streaks
can send messages to users

user:
thru the admin, each individual user will receive messages
sometimes customized, sometimes notification

flask: 
the glue between the sqlite database and the admin/user
